=== mobile.de Plugin ===
Requires at least: 3.6
Tested up to: 4.7.5

Ihr Fahrzeugbestand, den Sie in mobile.de pflegen und verwalten wird automatisch in Ihre WordPress Datenbank importiert.

== Description ==
Durch das neue WordPress Plugin zur direkten Anbindung Ihres Fahrzeugbestandes von mobile.de wird Ihre Website zum Showroom Ihres Autohandels. Ihr Fahrzeugbestand, den Sie in mobile.de pflegen und verwalten wird automatisch (in regelmäßigen, wählbaren Intervallen) in Ihre WordPress Datenbank importiert. Da für den Besucher keine Abfrage an die mobile.de API erfolgt, werden Ihre Fahrzeuge direkt von Ihrem Server, bzw. Hosting abgerufen. Das bedeutet maximale Performance und Verfügbarkeit.

== Installation ==
Um das mobile.de WordPress Plugin nutzen zu können muss Ihr Server mit PHP ab der Version 5.4 ausgestattet sein. Momentan stellen die größeren Hoster von PHP 5.3 auf PHP 5.4 bzw. 5.5 um.

WordPress

Das mobile.de WordPress Plugin benötigt eine aktuelle Version von WordPress. Diese können Sie bei http://wpde.org/ kostenlos herunterladen.

Das Plugin wird regelmäßig erneuert und an die kommenden Versionen von WordPress angepasst werden.

Ihre WordPress Installation sollte im Internet über eine Domain bzw. Subdomain erreichbar sein.

Der Ordner includes/uploads sollte frei beschreibbar sein, um die Bilder von mobile.de auf Ihren Server, Ihr Webpack zu übertragen.

Grundlegene Kenntnisse

Um das Plugin zu nutzen sollten Sie sich bereits mit WordPress beschäftigt haben und den WordPress Codex kennen. Sollten Sie keinerlei Erfahrungen mit WordPress und seiner Funktionsweise haben, so raten wir dringend davon ab das Plugin ohne die Hilfe einer qualifizierten Agentur / eines Programmierers zu nutzen. Sollten Sie hier keinen kompetenten Ansprechpartner haben, so stellen wir gerne den Kontakt zu einem von uns qualifizierten Integrationspartner her. Bitte beachten Sie, dass diese Dienstleistung kostenpflichtig ist.

I. Download
Sobald Sie das Plugin gekauft haben, können Sie sich die ZIP Datei mit allen Plugindaten herunterladen.

II. Installation
Installieren können Sie das Plugin wie jedes andere über das Admin Backend von WordPress. Hier bei Plugins auf Installieren klicken und die ZIP Datei auswählen.

Im nächsten Schritt erscheint unter Plugins -> mobile.de Plugin Lizenz. Bitte geben Sie hier Ihren Lizenzschlüssel ein, den Sie im Vorfeld per E-Mail erhalten haben.

Sobald Sie die Lizenz aktivieren, sollte ein grünes “Aktiviert” erscheinen. Sollte dies nicht der Fall sein, so prüfen Sie bitte den Lizenzschlüssel und vergewissern Sie sich bei einem Demo Download, das die Lizenz noch gültig ist.

Sobald Sie die Lizenz aktiviert haben, erscheint in Ihrem WordPress Backend unter Einstellungen ein neuer Menüpunkt “mobile.de WordPress Plugin”.

III. Einstellungen
Tragen Sie bei Einstellungen -> mobile.de WordPress Plugin die Zugangsdaten zur mobile.de Inserats Einbindung ein. Diese können Sie über Ihren Händlerbereich bei mobile.de unter Bestand -> Händler Werbung und Services -> Inserats-Einbindung aktivieren. 

Die mobile.de Api ist in Deutschland “https://services.mobile.de/”
Weiter unten haben Sie zahlreiche Funktionen um die zu importierenden Fahrzeuge zu selektieren. Sobald Sie alle Einstellungen nach Ihren Wünschen getroffen haben, können Sie Ihren Fahrzeugbestand mit einem Klick auf Fahrzeuge importieren in Ihre WordPress Datenbank laden. Dies kann je nach Fahrzeugbestand einige Zeit dauern.

IV. Fahrzeuge
Das Plugin legt für Sie automatisch den Menüpunkt Fahrzeuge im WordPress Backend an. Hier finden Sie eine Liste aller importierten Fahrzeuge. Sie können sich so einen Überblick über die importierten Fahrzeuge machen und diese nach belieben verändern.

V. Sidebar Widget
Unter Design -> Widgets finden Sie nun das mobile.de WordPress Plugin Sidebar Widget. Hier können Sie die anzuzeigenden Felder wählen und dem Kunden so die Möglichkeit geben, die Suche nach bestimmten Fahrzeugen einzuschränken. Wählen Sie mit Hilfe der Checkboxen die Filterkriterien, die Sie anbieten möchten. Dies ist vor allem sinnvoll, wenn Sie ein größeres Sortiment an Fahrzeugen anbieten.

VI. Ausgabeseite einrichten
Um eine Seite für Ihre Kunden mit Ihrem Fahrzeugbestand anzulegen, erstellen Sie einfach die Seite mit dem Namen “Fahrzeuge”. WordPress erkennt dann automatisch den Slug und zeigt dem Kunden hier den Fahrzeugbestand. Bitte achten Sie darauf, dass Sie unter Einstellungen -> Permalinks die Option “Beitragsname” gewählt haben.

VII. Anpassungen
Wir haben bereits versucht Ihnen soviel Arbeit wie möglich abzunehmen, dies ist jedoch im Hinblick auf die Gestaltungsvielfalt von Webseiten sehr schwierig. Deshalb sollte das Plugin in jedem Fall auf Ihr Template angepasst werden. Da die Fahrzeuge als Custom Post Type in WordPress integriert werden ist es für Ihre Werbeagentur ein leichtes, diese zu bearbeiten und perfekt in Ihrer Seite zu integrieren. Sollten Sie hierzu Fragen haben oder Hilfe benötigen, so stehen wir Ihnen jederzeit gerne zur Verfügung.
